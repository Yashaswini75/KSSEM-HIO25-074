# Smart Loan Assistant (Streamlit prototype)
## Quick start
1. Install Python 3.8+.
2. Install Tesseract (system binary) and ensure `tesseract` is on PATH.
3. Create a virtualenv and install dependencies:
 ```bash
#pip install -r requirements.txt
 ```
4. Run the app:
 ```bash
#streamlit run app.py
 ```
Files are persisted in the `data/` folder as CSVs. Sample banks are preseeded in `data/banks.csv`
